# Customer Segmentation Dataset Dictionary

This document details the datasets generated and utilized throughout the customer segmentation process. All files are located in the `customer_segmentation/datasets/` folder.

| Dataset Name | Source/Generated By | Description | Key Columns |
| :--- | :--- | :--- | :--- |
| **customer_features.csv** | `engineer_features.py` | Raw, unscaled customer features extracted from transactional records. | `customer_id`, `recency`, `frequency`, `monetary`, `average_purchase_value`, `customer_tenure`, `weekend_sales_ratio`, `active_months` |
| **customer_features_cleaned.csv** | `feature_audit.py` | Audited version of the customer features with infinite values handled and outliers audited. | Same as above with null/infinite values resolved. |
| **customers_features_no_refunds.csv** | `remove_refunds_and_recluster.py` | Intermediate features dataset containing only active customers after excluding refund-dominated/cancellation-heavy accounts. | Same as `customer_features.csv` |
| **customers_features_finalone.csv** | `remove_refunds_and_recluster.py` | Finalized cleaned feature dataset representing active customer behavior (no refund anomalies). | Same as `customer_features.csv` |
| **clustering_dataset_std.csv** | `prepare_segmentation_dataset.py` | Standardized dataset (using `StandardScaler`) prepared from the initial customer base. | Log-transformed and scaled versions of the features. |
| **clustering_dataset_rob.csv** | `prepare_segmentation_dataset.py` | Robustly scaled dataset (using `RobustScaler`) prepared from the initial customer base. | Robustly scaled versions of the features. |
| **clustering_dataset_std_v2.csv** | `remove_refunds_and_recluster.py` | Final standardized dataset used for both finalized K-Means clustering and DBSCAN validation. | Log-transformed and `StandardScaler` scaled features (excluding refund anomalies). |
| **correlation_matrix.csv** | `feature_audit.py` | Tabular correlation matrix of the features used to assess collinearity. | Feature names as rows and columns with Pearson correlation coefficients. |
| **customer_segments_kmeans.csv** | `kmeans_segmentation.py` | Output of initial K-Means clustering including the refund-dominated cluster. | `customer_id`, `cluster` |
| **customer_segments_kmeans_finalone.csv** | `remove_refunds_and_recluster.py` | Finalized K-Means segment assignments for active customers. | `customer_id`, `cluster`, `segment_name`, `recency`, `frequency`, `monetary`, etc. |
| **customer_segments_dbscan.csv** | `dbscan_analysis.py` | Validation cluster/outlier assignments computed via DBSCAN. | `customer_id`, `dbscan_cluster` (where `-1` denotes identified outliers) |
